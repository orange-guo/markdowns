    ClojureScript plumbing for your self-hosted REPLs.

  • awesome-clojure

Web​

  • electric
    a reactive Clojure dialect for web development that uses a compiler to infer the frontend/backend boundary
  • electric-fiddle
    Easiest way to get started with Electric Clojure
  • fulcro-electric-template
    A sample project that could be used as a starting point for a Fulcro 3 + Electric project.
  • ring
    Clojure HTTP server abstraction
  • ring-middleware
  • ring-json
    Ring middleware for handling JSON
  • compojure
    A concise routing library for Ring/Clojure
  • http-kit
    Simple, high-performance event-driven HTTP client+server for Clojure

Serialization​

  • nippy
    The fastest serialization library for Clojure
  • data.json
    JSON in Clojure

Data access​

  • toucan
    A classy high-level Clojure library for defining application models and retrieving them from a DB

Others​

  • joker
    Joker is a small interpreted dialect of Clojure written in Go. It is also a Clojure(Script) linter.
  • planck
    Planck is a stand-alone ClojureScript REPL for macOS and Linux.
    Planck launches instantly and is useful for scripting.
  • closh
    Bash-like shell based on Clojure
  • lumo
    Fast, cross-platform, standalone ClojureScript environment
  • cheshire
    Clojure JSON and JSON SMILE (binary json format) encoding/decoding
  • hugsql
    A Clojure library for embracing SQL.
  • honeysql
    Turn Clojure data structures into SQL
  • leiningen
    Moved to Codeberg; this is a convenience mirror
  • clj-ldap
    Clojure LDAP client
  • tools.cli
    Command-line processing
  • CLJ Easy
    This organization was created to collaborate on small utilities or documentation projects that are of general use for the Clojure community and aren't large enough to be hosted in a separate organization like babashka or clj-kondo.
  • graal-build-time
    Library to initialize Clojure packages at build time with GraalVM native-image.
  • babashka
    Native, fast starting Clojure interpreter for scripting.
